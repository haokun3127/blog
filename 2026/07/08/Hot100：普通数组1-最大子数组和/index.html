Hot100：普通数组1-最大子数组和

发表于2026-07-08|更新于2026-07-08|算法

题目

给你一个整数数组 nums，请找出一个具有最大和的连续子数组，并返回其最大和。

参考入口：

  • 力扣题目：53. 最大子数组和

复习导览

这道题要抓住一句话：以 nums[i] 结尾的最大子数组和，只取决于“前面的和还值不值得继续带上”。

复习点 内容
题型 贪心 / 一维动态规划
推荐解法 Kadane 算法
关键变量 current、ans
时间复杂度 O(n)
空间复杂度 O(1)
最容易错的点 current 和 ans 都应该从 nums[0] 开始，不能从 0 开始，否则全负数组会出错。

破题思路

如果前面的连续和是负数，那它只会拖后腿；如果是正数，就值得和当前元素拼起来。因此遍历到每个位置时，只需要在“单独从自己开始”和“接在前面后面继续累加”之间取最大值。

核心流程

  1. 初始化 current = nums[0]，ans = nums[0]。
  2. 从下标 1 开始遍历数组。
  3. 计算以当前位置结尾的最大和：要么只取当前元素，要么接上前面的最优结果。
  4. 用 ans 维护全局最大值。
  5. 遍历结束后返回 ans。

复杂度分析

  • 时间复杂度：O(n)。
  • 空间复杂度：O(1)。

易错点

  • current 和 ans 都应该从 nums[0] 开始，不能从 0 开始，否则全负数组会出错。
  • 这里求的是“连续子数组”，不能随便跳过中间元素。

一句话总结

最大子数组和的核心不是死记代码，而是先抓住“以 nums[i] 结尾的最大子数组和，只取决于“前面的和还值不值得继续带上”。”这条主线，再把模板写稳。
