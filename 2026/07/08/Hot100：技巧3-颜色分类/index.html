Hot100：技巧3-颜色分类

发表于2026-07-08|更新于2026-07-08|算法

题目

给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums，原地对它们进行排序，使得相同颜色的元素相邻，并按照红、白、蓝顺序排列。通常用 0、1、2 表示三种颜色。

参考入口：

  • 力扣题目：75. 颜色分类

复习导览

这道题要抓住一句话：一次扫描时，把 0 往左区放、把 2 往右区放，剩下中间自然就是 1。

复习点 内容
题型 双指针 / 原地交换
推荐解法 荷兰国旗问题
关键变量 left、right、i
时间复杂度 O(n)
空间复杂度 O(1)
最容易错的点 遇到 2 交换到右边后，i 不能立刻后移，因为换回来的新值还没检查。

破题思路

用三个指针把数组划成四段：左边是已经放好的 0 区，中间是待处理区，右边是已经放好的 2 区。遍历时根据 nums[i] 的值决定交换到哪一边。

核心流程

  1. 初始化 left = 0，right = n - 1，i = 0。
  2. 如果 nums[i] == 0，与 left 交换，并同时右移 left 和 i。
  3. 如果 nums[i] == 2，与 right 交换，并左移 right。
  4. 如果 nums[i] == 1，直接 i++。
  5. 循环直到 i > right。

代码

复杂度分析

  • 时间复杂度：O(n)。
  • 空间复杂度：O(1)。

易错点

  • 遇到 2 交换到右边后，i 不能立刻后移，因为换回来的新值还没检查。
  • 这题要求原地排序，不要直接调用库排序。

一句话总结

颜色分类的核心不是死记代码，而是先抓住“一次扫描时，把 0 往左区放、把 2 往右区放，剩下中间自然就是 1。”这条主线，再把模板写稳。
