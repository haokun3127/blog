Hot100：二分查找1-搜索插入位置

发表于2025-10-30|更新于2026-05-30|算法

题目

给定一个排序数组 nums 和一个目标值 target，在数组中找到目标值，并返回其索引。

如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

要求使用时间复杂度为 O(log n) 的算法。

示例：

提示：

  • 1 <= nums.length <= 10^4
  • nums 是升序数组，且没有重复元素。

复习导览

这道题要抓住一句话：二分查找结束后，left 就是目标值应该插入的位置。

复习点 内容
题型 二分查找
关键条件 数组有序
推荐写法 while (left <= right)
找到目标 返回 mid
没找到目标 返回 left
时间复杂度 O(log n)
最容易错的点 不理解为什么最后返回 left

破题思路

题目有两个任务：

  1. 如果 target 存在，返回它的下标。
  2. 如果 target 不存在，返回它应该插入的位置。

因为数组是有序的，所以可以用二分查找。

二分查找每次看中间值 nums[mid]：

  • 如果 nums[mid] == target，直接返回 mid。
  • 如果 nums[mid] < target，说明目标在右边，移动 left。
  • 如果 nums[mid] > target，说明目标在左边，移动 right。

如果最后没有找到，循环结束时 left > right。此时 left 指向第一个大于 target 的位置，也就是插入位置。

用边界理解 left 和 right

二分过程中始终可以这样理解：

  • left 左边的元素都已经确认 < target。
  • right 右边的元素都已经确认 > target。

当循环结束时，left 和 right 交错：

这时 left 就站在“第一个不小于 target 的位置”上。

本题数组没有重复元素，所以这个位置就是：

  • 如果存在目标值，会在循环中提前返回。
  • 如果不存在目标值，就是插入位置。

三种插入情况

情况 示例 返回值
插入到开头 [1,3,5,6], target = 0 0
插入到中间 [1,3,5,6], target = 2 1
插入到末尾 [1,3,5,6], target = 7 4

复杂度分析

  • 时间复杂度：O(log n)。每次都把搜索区间缩小一半。
  • 空间复杂度：O(1)。只使用 left、right、mid 几个变量。

搜索插入位置是二分查找的基础题。找到目标就返回 mid；找不到时，二分结束后的 left 正好停在目标应该插入的位置。
