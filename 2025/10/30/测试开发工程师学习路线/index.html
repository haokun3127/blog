📖 阶段1：入门

编程语言

Java（1-2个月）⭐

核心知识点

基础部分

  • 数据类型（基本数据类型、引用数据类型）
  • 流程控制（判断、循环）
  • 面向对象
    • 方法、类
    • 重载、重写
    • 封装、继承、多态
  • 数组、枚举
  • 抽象类、接口
  • 集合类
  • 常用类（String、日期时间）
  • 泛型、注解
  • 多线程
  • IO流、反射
  • 异常处理

  • 黑马Java基础（针对完全没有计算机基础的小白）

文档资料

  • 廖雪峰 Java 教程

  • IDEA Win 常用快捷键

推荐书籍

  • 《Head First Java》
  • 《JAVA核心技术(卷1)》

💼 经典面试题

  • 为什么重写 equals 还要重写 hashcode？
  • == 和 equals 比较的区别？
  • final 关键字的作用？
  • 介绍 Java 的集合类
  • ArrayList 和 LinkedList 的区别？
  • 多态如何实现，重载和重写的区别？

Python（1个月）⭐

核心知识点

基础部分（14天）

  • Python 安装
  • 开发工具（PyCharm、Sublime、VS Code）
  • 变量（定义变量、关键字、命名规则、基本数据类型、类型转换）
  • 运算符和表达式
  • 流程控制（条件分支、循环）
  • 基本数据结构
    • 字符串、列表、元组、集合、字典
  • 函数
    • 定义、参数传递、作用域
    • lambda 表达式
    • 常用内置函数
  • 面向对象编程
    • 类和对象
    • 三大特性（封装、继承、多态）
    • 装饰器、反射
  • 模块与包
  • 异常处理
  • 文件操作

进阶部分（20天）

  • 函数进阶（闭包、匿名函数、生成器函数、装饰器、高阶函数）
  • 正则表达式
  • 数据库编程（数据库基础、SQL 编写、查询、事务、数据库设计、数据库调优）
  • 并发编程（同步异步、阻塞非阻塞、多线程、多进程、协程）
  • 网络编程（网络基础、网络协议、WebSocket）

常用库

  • Requests：网络请求
  • openpyxl：Excel 读写库
  • nose：测试工具
  • faker / datafaker：构造测试数据

  • 千锋教育 700 集零基础 Python 教程（非常全面，基础、web 开发、爬虫、数据分析、AI 基础都讲了）
