学生管理系统（基础版）

学生管理系统实现步骤

案例需求

针对目前我们的所学内容，完成一个综合案例：学生管理系统。该系统主要功能如下：

  • 添加学生：通过键盘录入学生信息，添加到集合中
  • 删除学生：通过键盘录入要删除学生的学号，将该学生对象从集合中删除
  • 修改学生：通过键盘录入要修改学生的学号，将该学生对象其他信息进行修改
  • 查看学生：将集合中的学生对象信息进行展示
  • 退出系统：结束程序

实现步骤

步骤 1：定义学生类

包含以下成员变量：

  • private String sid - 学生 id
  • private String name - 学生姓名
  • private String age - 学生年龄
  • private String address - 学生所在地

步骤 2：学生管理系统主界面的搭建

  1. 用输出语句完成主界面的编写
  2. 用 Scanner 实现键盘输入
  3. 用 switch 语句完成选择的功能
  4. 用循环完成功能结束后再次回到主界面

步骤 3：添加学生功能

  1. 定义一个方法，接收 ArrayList<Student> 集合
  2. 方法内完成添加学生的功能
    • 键盘录入学生信息
    • 根据录入的信息创建学生对象
    • 将学生对象添加到集合中
    • 提示添加成功信息
  3. 在添加学生的选项里调用添加学生的方法

步骤 4：查看学生功能

  1. 定义一个方法，接收 ArrayList<Student> 集合
  2. 方法内遍历集合，将学生信息进行输出
  3. 在查看所有学生选项里调用查看学生方法

步骤 5：删除学生功能

  1. 定义一个方法，接收 ArrayList<Student> 集合
  2. 方法中接收要删除学生的学号
  3. 遍历集合，获取每个学生对象
  4. 使用学生对象的学号和录入的要删除的学号进行比较，如果相同，则将当前学生对象从集合中删除
  5. 在删除学生选项里调用删除学生的方法

步骤 6：修改学生功能

  1. 定义一个方法，接收 ArrayList<Student> 集合
  2. 方法中接收要修改学生的学号
  3. 通过键盘录入学生对象所需的信息，并创建对象
  4. 遍历集合，获取每一个学生对象，并和录入的修改学生学号进行比较，如果相同，则使用新学生对象替换当前学生对象
  5. 在修改学生选项里调用修改学生的方法

步骤 7：退出系统

使用 System.exit(0); 退出 JVM

学生管理系统（升级版）

升级版新增功能

在基础版的基础上，升级版添加了完整的用户认证系统，主要包括以下功能：

  • 用户注册：新用户可以注册账号
  • 用户登录：通过用户名、密码和验证码登录
  • 忘记密码：通过身份证和手机号验证后重置密码
  • 登录限制：3次登录失败后账号锁定

项目架构

升级版采用了分层架构设计：

  • App.java - 主程序入口，负责用户认证
  • User.java - 用户实体类
  • Student.java - 学生实体类
  • StudentSystem.java - 学生管理系统功能模块

用户实体类 (User.java)

核心功能详解

用户注册功能

注册流程：

  1. 用户名验证

    • 长度：3-15 位之间
    • 格式：字母 + 数字的组合（不能纯数字）
    • 唯一性：不能与已有用户重复

  2. 密码验证

    • 需要输入两次密码
    • 两次密码必须一致

  3. 身份证号码验证

    • 长度：18 位
    • 不能以 0 开头
    • 前 17 位必须是数字
    • 最后一位可以是数字或大写/小写 X

  4. 手机号码验证

    • 长度：11 位
    • 不能以 0 开头
    • 必须全部是数字

用户登录功能

登录流程：

  1. 输入用户名，验证用户是否存在
  2. 输入密码
  3. 输入验证码（不区分大小写）
  4. 验证用户名和密码是否正确
  5. 登录成功后进入学生管理系统

验证码生成机制：

  • 随机生成 4 个字母（大小写）
  • 在末尾添加 1 个随机数字
  • 将数字随机插入到验证码中的任意位置

登录限制机制：

  • 最多允许 3 次登录尝试
  • 3 次失败后账号锁定
  • 每次失败会提示剩余尝试次数

忘记密码功能

重置密码流程：

  1. 输入用户名，验证用户是否存在
  2. 输入身份证号码
  3. 输入手机号码
  4. 验证身份证和手机号是否与注册信息一致
  5. 两次输入新密码，确认一致后修改成功
