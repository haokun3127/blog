Hot100 题型地图

参考 LeetCode Hot100 的经典面试题定位，官方题单本身强调“经典、高频、按热门主题组织”。复习时可以按下面的题型分组推进，而不是只按发布时间刷。

题型 复习关键词 当前建议
哈希 HashMap、HashSet、分组、去重、补数 已开始，先把 3 题反复复盘
双指针 左右指针、快慢指针、原地移动 下一批适合接上
滑动窗口 窗口边界、计数表、最长 / 最短子串 和哈希一起复习效果好
子串 前缀和、窗口、字符串状态 注意边界和重复字符
普通数组 前缀和、差分、原地标记、排序 训练数组状态转移
矩阵 原地旋转、方向数组、层序遍历 重点看坐标变化
链表 虚拟头节点、快慢指针、反转 画指针变化图
二叉树 DFS、BFS、递归返回值 把遍历模板先写熟
图论 BFS、DFS、并查集、拓扑 先从网格图和岛屿类题入手
回溯 选择、撤销、剪枝 写清递归参数
二分查找 搜索区间、边界、插入位置 已开始，固定模板
栈 单调栈、括号匹配、路径处理 重点理解“维护什么顺序”
堆 TopK、优先队列、合并 关注比较器和堆大小
贪心 局部最优、区间、买卖股票 要能说明为什么贪心成立
动态规划 状态定义、转移方程、初始化 不急，后面集中刷

单题复习模板

每道题统一按这个格式复习：

模块 要回答的问题
题型识别 看到题目后，为什么想到这个解法？
核心变量 哈希表、左右指针、窗口、栈、DP 数组分别表示什么？
推导过程 用一个样例手推状态变化
代码模板 保留一份能默写的 Java 写法
易错点 记录边界、空输入、重复元素、返回值
复杂度 说清时间和空间复杂度来源
